# 抖音评论导出功能使用说明

## 📋 功能概述

本次更新为抖音下载项目增加了完整的视频评论导出功能，支持：
- ✅ 通过视频ID导出评论
- ✅ 通过视频URL自动解析并导出评论
- ✅ 在Web界面直接导出评论CSV文件
- ✅ 在视频解析结果页面直接下载评论
- ✅ 命令行工具支持
- ✅ API接口支持

## 🚀 快速开始

### 方式一：Web界面（推荐）

1. 访问项目主页：`http://localhost:8000`
2. 选择功能：
   - **💬导出视频评论 (通过ID)** - 直接输入视频ID导出评论
   - **💬导出视频评论 (通过URL)** - 输入视频URL，自动解析后导出评论
   - **🔍批量解析视频** - 解析视频后，在结果页面可直接下载评论

3. 操作步骤：
   - 输入视频ID或URL
   - 设置想要获取的评论数量（默认100条）
   - 点击提交
   - 等待评论获取完成
   - 点击"下载CSV文件"按钮保存

### 方式二：视频解析结果页面

当你解析一个抖音视频后，在结果页面会看到：

| 下载评论CSV |
|-------------|
| 点击下载     |

直接点击即可下载该视频的评论CSV文件。

### 方式三：命令行工具

```bash
# 基本使用
python3 comment_export_tool.py <aweme_id> [max_comments]

# 示例：导出100条评论
python3 comment_export_tool.py 7372484719365098803 100

# 指定输出文件路径
python3 comment_export_tool.py 7372484719365098803 100 /path/to/output.csv
```

### 方式四：API接口

```bash
# 直接下载评论CSV
curl -O "http://localhost:8000/api/douyin/comments/export?aweme_id=7372484719365098803&max_comments=100"

# 自定义文件名
curl -O "http://localhost:8000/api/douyin/comments/export?aweme_id=7372484719365098803&max_comments=100&filename=my_comments"
```

**API参数说明：**
- `aweme_id` (必需): 视频作品ID
- `max_comments` (可选): 最大评论数，默认100
- `filename` (可选): 自定义文件名（不含扩展名）

### 方式五：快速启动脚本

```bash
./export_comments.sh
```

按提示选择操作即可。

## 📊 导出的CSV格式

导出的CSV文件包含以下字段：

| 字段 | 说明 | 示例 |
|------|------|------|
| 评论人 | 评论者昵称 | 用户123 |
| 评论内容 | 评论的具体内容 | 这个视频太棒了！ |
| 点赞量 | 评论获得的点赞数 | 123 |
| 评论时间 | 评论发表时间 | 2024-01-15 10:30:00 |

**CSV文件特点：**
- 使用UTF-8-BOM编码，Excel可直接打开显示中文
- 包含表头，方便数据分析
- 支持大量数据导出（受API限制影响）

## 🔍 如何获取视频ID

### 方法一：从视频URL提取
```
直接复制视频URL中的数字部分：
https://www.douyin.com/video/7372484719365098803
                                       ↑
                                    这是aweme_id
```

### 方法二：从分享链接获取
1. 复制抖音APP中的分享链接（如：https://v.douyin.com/e4J8Q7A/）
2. 使用"💬导出视频评论 (通过URL)"功能，会自动解析出视频ID
3. 或者在解析结果页面直接下载评论

### 方法三：使用API解析
```bash
curl "http://localhost:8000/api/hybrid/video_data?url=https://v.douyin.com/e4J8Q7A/"
```
返回结果中的 `video_id` 或 `aweme_id` 字段就是视频ID。

## 💡 功能特点

### 1. 自动分页获取
- 无需手动处理分页
- 自动获取所有评论（直到达到设置的最大数量或无更多评论）
- 内置请求间隔，避免被风控

### 2. 智能过滤
- 自动过滤无效评论
- 解评论人信息、内容、点赞数、时间等完整数据

### 3. 多种导出方式
- Web界面导出（适合非技术用户）
- API导出（适合开发者）
- 命令行工具（适合批量处理）
- 解析结果快捷下载（适合单次使用）

### 4. 数据完整性
- 获取评论人昵称
- 获取完整评论内容
- 获取点赞数量
- 获取发表时间

## ⚠️ 注意事项

1. **请求频率限制**
   - 工具内置了0.5秒的请求间隔
   - 建议不要设置过大的评论数量（如超过1000条）
   - 大量评论获取需要较长时间

2. **Cookie配置**
   - 确保项目配置文件中的Cookie有效
   - Cookie失效可能导致无法获取评论
   - 建议定期更新Cookie

3. **视频类型限制**
   - 当前仅支持抖音视频评论导出
   - TikTok和Bilibili评论导出暂未实现

4. **数据准确性**
   - 评论数据来源于抖音API
   - 可能存在API返回的数据延迟或不完整
   - 部分评论可能已被删除或隐藏

5. **时间显示**
   - 当前时间显示可能存在时区问题
   - 不影响评论数据的其他内容

## 🛠️ 故障排除

### 问题1：无法获取评论数据

**可能原因：**
- Cookie无效或过期
- 网络连接问题
- 视频已被删除
- 视频ID错误

**解决方案：**
1. 检查配置文件中的Cookie是否有效
2. 测试网络连接
3. 确认视频ID是否正确
4. 检查服务器日志文件：`tail -f app.log`

### 问题2：CSV文件显示乱码

**解决方案：**
- 文件使用UTF-8-BOM编码，Excel应该能正常显示
- 如果Excel中仍然乱码，可尝试：
  - 用记事本打开后另存为ANSI编码
  - 使用WPS、Google Sheets等其他软件打开

### 问题3：无法访问Web界面

**解决方案：**
1. 确认服务器正在运行：`ps aux | grep python3`
2. 检查端口是否被占用：`lsof -i:8000`
3. 查看服务器日志：`tail -f app.log`
4. 重启服务器：`python3 start.py`

### 问题4：评论数量不完整

**可能原因：**
- 视频评论总数不足
- API返回的评论有限制
- 部分评论被抖音过滤

**解决方案：**
- 尝试增大max_comments参数
- 分多次导出，合并结果
