# 抖音_TikTok_下载工具 PRD

## 1. 产品概述

### 1.1 产品简介
一款支持抖音、TikTok、Bilibili 三大平台的视频解析与评论导出工具。

### 1.2 产品定位
面向内容创作者、社媒运营人员及数据分析师的批量视频下载与评论数据采集工具。

### 1.3 目标用户
- 内容创作者（采集参考素材）
- 社媒运营人员（监控竞品评论）
- 数据分析师（评论数据挖掘）

---

| 层级 | 技术栈 |

| 前端 | Vue 3 + Vite 5 + TypeScript + Tailwind CSS + Pinia |
| 后端 | Python 3 + FastAPI + PyWebIO |

## 3. 页面功能说明

### 3.1 首页 `/`

**功能：** 平台入口页面，提供四大功能模块的导航入口。

**UI布局：**
- Logo + 标题区
- 四张功能卡片（2x2网格）
- 页脚信息

**功能卡片：**

| ID | 图标 | 标题 | 跳转路由 |
|----|------|------|----------|
| parse | 🔍 | 批量解析视频 | `/parse` |
| export-douyin-id | 💬 | 导出抖音评论 | `/export/douyin` |
| export-tiktok-id | 🎬 | 导出TikTok评论 | `/export/tiktok` |
| download-history | 📥 | 导出记录 | `/download-history` |

** Modal功能：**
- iOS Shortcuts 使用指南
- API 文档跳转
- About 版本信息

---

### 3.2 批量解析页 `/parse`

**功能：** 批量解析并下载无水印视频/图集。

**支持平台：**
- 抖音 (Douyin)
- TikTok
- Bilibili

**输入：**
- 多行文本框，最多输入 20 个链接
- 支持分享链接和直链混合输入

**处理流程：**
1. URL 验证
2. 逐个解析（显示实时进度：已解析/总数）
3. 实时展示每个链接的解析结果

**输出展示：**

| 状态 | 显示内容 |
|------|----------|
| 解析成功-视频 | 平台、描述、作者、下载按钮 |
| 解析成功-图集 | 平台、描述、作者、图片预览网格 |
| 解析失败 | 错误原因（红色提示） |

**统计栏：**
- 成功数（绿色）
- 失败数（红色）
- 总数（蓝色）

---

### 3.3 抖音评论导出页 `/export/douyin`

**功能：** 导出抖音视频评论到 CSV 文件。

**输入方式（二选一）：**
1. **通过URL导出** - 输入视频分享链接
2. **通过ID导出** - 输入视频 ID

**配置参数：**
- 最大评论数：1-100（默认100）

**评论字段：**

| 字段 | 说明 |
|------|------|
| user_nickname | 评论者昵称 |
| text | 评论内容 |
| digg_count | 点赞数 |
| reply_count | 回复数 |
| create_time | 评论时间 |
| ip_label | IP 属地 |
| comment_id | 评论 ID |

**操作流程：**
1. 输入视频 URL/ID → 点击「获取评论」
2. 预览前 5 条评论
3. 点击「导出CSV」创建导出任务
4. 跳转至导出记录页查看进度

---

### 3.4 TikTok评论导出页 `/export/tiktok`

**功能：** 导出 TikTok 视频评论到 CSV 文件。

**功能与 `/export/douyin` 完全一致，仅平台不同。**

| 差异点 | 抖音 | TikTok |
|--------|------|--------|
| 平台标识 | 抖音 | TikTok |
| API 端点 | `/api/douyin/comments/*` | `/api/tiktok/comments/*` |

---

### 3.5 导出记录页 `/download-history`

**功能：** 管理所有评论导出任务，支持进度监控和文件下载。

**任务状态：**

| 状态 | 标识 | 说明 |
|------|------|------|
| pending | ⏳ 蓝 | 等待开始 |
| running | 🔄 蓝 | 下载中（显示进度条） |
| completed | ✅ 绿 | 已完成 |
| failed | ❌ 红 | 失败（显示错误信息） |

**列表字段：**
- 平台类型（抖音/TikTok）
- 视频 ID (aweme_id)
- 评论总数
- 创建时间（相对时间）
- 任务状态

**操作：**

| 操作 | 按钮颜色 | 条件 |
|------|----------|------|
| 下载原始评论 | 绿色 | 任务已完成 |
| AI评论分类 | 紫色 | 任务已完成且未分类 |
| 下载已分类评论 | 紫色 | 分类已完成 |
| 删除 | 红色描边 | 任意状态 |
| 批量删除 | - | 选中多个任务后 |

**AI 分类功能：**
- 对评论进行情感分类（正面/负面/中性）
- 显示分类统计摘要
- 支持下载分类后的 CSV

**实时更新：**
- 页面加载时获取所有任务
- 若存在 `running` 状态任务，每 5 秒轮询更新

## 5. API 概览

| 方法 | 端点 | 功能 |
|------|------|------|
| GET | `/api/douyin/comments/fetch` | 获取抖音评论预览 |
| GET | `/api/tiktok/comments/fetch` | 获取TikTok评论预览 |
| POST | `/api/tasks/comments` | 创建导出任务 |
| GET | `/api/tasks/comments/tasks` | 获取所有任务 |
| GET | `/api/tasks/comments/{task_id}` | 获取单个任务状态 |
| GET | `/download/comments/{filename}` | 下载CSV文件 |

## 6. 配置文件

`config.yaml` 控制以下参数：
- 最大并发解析数
- 每批最大 URL 数量
- API 超时时间
- UI 主题配置
- Easter egg 开关
